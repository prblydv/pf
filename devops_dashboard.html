Software Project

Finance Management Application

Built with Java + JHipster · CI/CD on GitHub · Role: Scrum Master

A collaborative capstone by 4 BSc students focused on simplifying personal finance with real-time analytics, goal tracking, and expense monitoring — all in one seamless platform.

System Glimpses

Spending Summary View

Shows weekly income, expenses, and budget insights using circular indicators. Recent expenses listed with breakdown.

Dashboard Weekly Stats

Highlights weekly usage trends across days with responsive graph lines — helps users identify peak spending periods.

Category Breakdown

Displays categorized expenses with color-coded UI and quick insights for each spending group.

Problem

Managing personal finances often feels scattered — users juggle multiple apps for budgeting, tracking income, and visualizing spending. Existing apps are either too simplistic or too complex for everyday users. Our goal was to deliver a clean, actionable dashboard that simplifies financial decision-making.

Approach

We built the application using the JHipster generator, which scaffolded the project into a modular monorepo with key folders like .jhipster, src, webpack, and .mvn for Maven build support. The backend was generated in Java with Spring Boot, while the frontend used React.js with Webpack bundling.

We used Node.js and majh.js for deployment scripts and automation logic. Continuous Integration and Deployment (CI/CD) was implemented via .gitlab-ci.yml, ensuring every commit triggered linting, testing (via jest), and auto-deployment. Configuration files such as cypress.config.ts and ngsw-config.json enabled E2E testing and PWA support.

I served as the Scrum Master for our 4-member BSc team — organizing sprint planning, daily standups, and retrospectives. We followed a strict Git-based agile workflow using feature branches, pull requests, and GitHub Projects for tracking progress.

What I Delivered

  • Led agile ceremonies and tracked progress using GitHub Projects.
  • Integrated CI/CD workflows for automated builds and deployments.
  • Optimized dashboard UX for mobile and desktop devices.
  • Ensured data integrity with backend validations and modular design.

Impact

The app was successfully deployed with zero downtime across sprints. We achieved smooth budget tracking, intuitive category-based analytics, and interactive reporting — helping users stay within budget 90% of the time, based on test user feedback.
